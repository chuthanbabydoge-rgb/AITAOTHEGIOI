Đây là dự án đang hoạt động.

TUYỆT ĐỐI KHÔNG:

* Xóa file cũ
* Xóa dữ liệu cũ
* Xóa save cũ
* Xóa tab UI cũ
* Xóa dashboard cũ
* Xóa lịch sử thế giới
* Đổi tên file cũ
* Đổi tên class cũ
* Đổi tên biến cũ
* Thay thế hệ thống cũ
* Ghi đè logic cũ

* Chỉ được mở rộng
* Chỉ được thêm code
* Chỉ được tích hợp

EXPAND ONLY

NEVER DELETE

NEVER REPLACE

NEVER REBUILD
